Course Project

Use the guidelines (available by links next to each item) to prepare a written report for the course project. Send the course project report and the presentation (created using Microsoft PowrPoint or other presentations tool) to the Teacher by the end of semester. Await for the course project defense appointment.

  • Select the topic of your course project from the list (UKR | ENG) and inform the Teacher.
  • Complete the steps given in the course project template (UKR | ENG) including at least the paragraph 2.1.
  • Use the guidelines (given below next to each item) to develop the database application as part of your course project (OPTIONAL):
    • Full-stack development using MySQL, PHP (or Java, Python, .NET, NodeJS, etc.), and HTML/CSS/JavaScript (UKR | ENG)
    • No-code (Low-code) development using the Oracle APEX platform (UKR | ENG)
