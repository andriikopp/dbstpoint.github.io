Course Project

Use the guidelines (available by links next to each item) to prepare a written report for the course project. Send the course project report and the presentation (created using Microsoft PowrPoint or other presentations tool) to the Teacher by the end of semester. Await for the course project defense appointment.

  • Select the topic of your course project from the list (UKR | ENG) and inform the Teacher. Attention! You can make the project based on your laboratory works, but the maximum score will not be greater than 75.
  • Complete the steps given in the course project template (📥 download UKR | ENG) including at least the paragraph 2.1. Attention! Prepare the handwritten project (including at least the paragraph 2.1) and send photos if you do not have a computer.
  • (*optional: + EXTRA points) Use the guidelines (given below next to each item) to develop the database application as part of your course project:
    • Full-stack development using MySQL (or MS SQL Server, PostgreSQL, SQLite, MongoDB, etc.), PHP (or Java, Python, .NET, NodeJS, etc.), and HTML/CSS/JavaScript:
      • Development of an elementary PHP database web application (UKR | ENG)
      • 🔥 The example of an elementary PHP database web application (📥 download)
      • 🔥 Case-study: Simple online shop/mart (PHP & MySQL step-by-step video guide) - Part 1 UKR | ENG
      • 🔥 Case-study: Simple online shop/mart (PHP & MySQL step-by-step video guide) - Part 2 UKR | ENG
      • ⭐ Software version control using Git (UKR | ENG)
      • ⭐ Advanced web application development using MySQL, PHP, HTML, CSS and JavaScript (UKR | ENG)
    • No-code (Low-code) development using the Oracle APEX platform:
      • Development of an elementary Oracle APEX database web application (UKR | ENG)
      • 🔥 Case-study: Grocery store database and application (Oracle APEX step-by-step video guide) UKR | ENG
